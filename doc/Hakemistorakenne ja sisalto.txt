TilastoKP
	doc - Dokumentaatio		
	src - Kähdekoodit
		app - kaikki AngularJS + front end -koodit
			css - Tyylitiedostot
			scripts - AngularJS-koodit. Jos jotain alihakemistoa ei tarvita, poistetaan myöhemmin.
				controllers
					TeamController.js
						- Ohjaa joukkueen tietoja.
					PlayerController.js
						- Pelaajan lisääminen ja muokkaaminen.
					MatchController.js
						- Ottelun ohjaus
				directives
				factories
					- MatchFactory.js
						- Tietojen välitys käyttöliittymän (kontrollerien kautta) ja tietokannan väillä.
						- Esimerkiksi pelaajatiedot, tapahtumatiedot, jne.
						- Käytetään aluksi tätä yhtä factorya ja jos kasvaa liikaa, pilkotaan useammaksi.
				services
			views - Näkymät
				index.html
					- Etusivu
					- Näytetään kaikkien muiden sivujen yhteydessä paitsi ottelusivulla (game.html)
				team.html
					- Joukkuen nimen tulostus (joukkuevalikko myöhemmin jos aikaa)
					- Joukkueet: joukkueiden käsittely (toteutetaan myöhemmin jos aikaa)
				player_list.html
					- Pelaajalista
				start_match.html
					- Ottelun aloitus
					- Täältä pääsee ottelusivulle (match.html)
				match.html
					- Ottelusivu
				reports.html
					- Edellisten otteluiden näkymät
					- Täältä pääsee ottelusivulle, joka on muokattu raporttia varten.
		server - Back end -koodit (NodeJS jne.)
